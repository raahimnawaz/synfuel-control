# Edge inference benchmark

The Phase 3 surrogate, deployed as a hand-rolled C++ forward pass with the trained
weights baked into the binary (`include/weights.h`, generated by `python -m pinn.train`).
Zero runtime dependencies — no Python, no ONNX Runtime, no BLAS — so it drops onto
MCU-class / edge hardware directly.

**Target architecture:** NVIDIA Jetson Orin Nano. **Benchmarked on host** (Apple Silicon
laptop, `clang++ -O3`, single-precision). Treat the absolute numbers as host figures; the
point is the *ratio* and that inference is negligible in the control-loop budget.

## Results

| Engine | Per-inference latency | Throughput | Notes |
|---|---|---|---|
| **Hand-rolled C++** (`synfuel_edge`) | **0.98 µs** | ~1.0 M/s | zero deps, weights baked in |
| ONNX Runtime (Python, single sample) | 4.71 µs | ~0.21 M/s | reference path |

**Speedup: ~4.8×** over the Python ONNX Runtime reference for single-sample inference
(the relevant case for real-time control, which evaluates one operating point at a time).

**Numerical parity:** C++ vs. ONNX max abs difference **6.4 × 10⁻⁵** over 200 random
inputs — i.e. identical to single-precision round-off. The C++ port is a faithful
reproduction of the trained model, not an approximation.

## Honest framing of the latency

~1 µs is the *bare inference* time. It is **not** the closed-loop control period — that is
millisecond-scale, set by the reactor's thermodynamic time constants and by sensor/serial
latency (Phase 6), not by the model. The value of this phase is twofold:

1. it shows the **deployment path** (trained PyTorch model → portable, dependency-free
   native binary), and
2. it confirms inference is **negligible** in the loop's time budget — the controller can
   run far faster than the plant needs, leaving headroom for everything else.

## Reproduce

```bash
# Build (canonical):
cmake -S edge -B edge/build -DCMAKE_BUILD_TYPE=Release && cmake --build edge/build
./edge/build/synfuel_edge --bench 2000000

# Or directly:
clang++ -std=c++17 -O3 -Iedge/include edge/src/main.cpp edge/src/mlp.cpp -o edge/synfuel_edge
echo "3 25 490 1 60000" | ./edge/synfuel_edge      # -> predicted 6-state
```
